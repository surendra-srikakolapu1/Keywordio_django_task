1 . Make sure that python is installed

2 . Move to the manage.py directory and open command prompt here

type the command to activate virtual environmnet

* ./env/Scripts/activate

3 . Install requirements.txt with this command

* pip3 install -r requirements.txt

4 . After successful installation , run the server at same directory

* python manage.py runserver

5 . Open http://127.0.0.1:8000/ in browser to check application.

6. Register and login for further activites.
